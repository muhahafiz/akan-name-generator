# Setup Instructions
1.Clone this page locally by running `git clone git@github.com:muhahafiz/akan-name-generator.git`
2.After cloning `cd akan-name-generatror` and run `code .` to open the project on Visual Studio Code.
3.Make sure you have {Live Server} `https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer`extension installed
4.Click the go live butto n on the bottom of the screen bar of VS Code as shown below 
![VS Code screenshot](images/Golive.png) to see the web application
The akan-name-generator website should open.
5.Play to find out
